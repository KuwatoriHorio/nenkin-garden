# 設計メモ — netphys（網 Physarum・移動性フォアジング前進波）候補案

**位置づけ**: これは**判断材料の設計メモ（C案・コードなし）**。ユーザー観察（2026-07-11）の
「数箇所から扇状に広がり→衝突で網化→限界で外側数点を新成長点に選び→そこまでの経路を残して縮退し
→再び扇状に広がる、を繰り返して移動する」Physarum の前進波を、実装可能なレベルに具体化する。
§0 の設計軸に関わる**新モデル**（Jones でも tree でもない第3のモデル）なので、作るか否かは人間が決める（§8）。

> **確定済み（ユーザー決定）**: ハイブリッド（探索前線＋網）で進める／**網（ループ）表現は必須**／
> **限界トリガ＝周期式**（一定 tick ごとに consolidation）／**新成長点＝最外周ノード**。
> **前提**: 現行 Jones モデル・tree-growth-001/002（別実験・完了）は無傷で残す。netphys は独立新コア。

---

## 0. これまでの決定の反映と supersede（混乱回避）
tree モデル向けの過去決定のうち、網 Physarum で**引き継ぐ / 置き換わる**もの:
- 引き継ぐ: **保存量（原形質）は保存的に再配分**（探索で使い→刈り込みで戻す＝後方から前方へ translocate）。
  **双方向フロー**＝ Kirchhoff の流れ（Tero）として自然に実現。**ソフト標高忌避・境界・決定性**。
- 置き換わる: 「親子のみの木」→ **一般グラフ（ループ可）**。「体積<しきいで分岐」（木のフォーク規則）→
  **探索前線＋衝突での融合(anastomosis)**。「局所/ホームまで退縮」→ **流れによる刈り込み（Tero）**。

## 1. データ構造（一般グラフ）
- **ノード** `{id, x, y}`（陸・標高場 E 上）。**辺** `{a, b, D, L}`（無向・コンダクタンス D＝管の太さ、長さ L）。
  **ループを許す一般グラフ**（既存 `analysis/graph.rs`・Kirchhoff が一般グラフ前提なので再利用可）。
- **原形質（保存量）**: `total_mass = free_budget + Σ(D_e · L_e)`（構造＝管に蓄えた分）。回収で増え、
  探索で free→管へ、刈り込みで管→free へ戻る（保存的）。
- **成長点（source）**: 現在アクティブな前線ノード集合。**砂糖**（置く/取る）は sink/source。
- 場は現行と共有（陸海マスク・E・`default_home`）。§0 の動詞（砂糖・時間）は不変。

## 2. サイクル（2タイムスケール）
毎 tick の**速い探索** ＋ 周期 N tick ごとの**遅い consolidation**。決定論・ノード/辺 id 昇順・単一 PRNG。

**Phase 1｜扇状探索（毎tick）**
各アクティブ前線ノードから、space colonization ＋ ランダム探索（tree-growth-002 の w_rand 発想を流用）で
外へ前線を伸ばし、ノード/辺を追加。複数フロントが扇状に広がる。生成は free_budget を消費（新辺は D 小）。

**Phase 2｜衝突で網化（anastomosis・探索中に判定）**
成長ノードが既存ノード/辺に**融合距離 d 以内**へ近づいたら**辺を追加してループを閉じる**（グリッド近接で
決定的に検出）。これが"網"を作る。過融合（塊化）／過疎（網にならない）を避ける d が要調整。

**Phase 3｜周期的 consolidation（N tick ごと・Tero 刈り込み）**
1. **新成長点＝最外周ノード**を選ぶ（コロニー重心から遠い上位 k 個／凸包頂点等・決定的）。
2. 最外周点（＋残存する砂糖 sink）を source/sink 端子として **Kirchhoff を解く**（`analysis/flow.rs` 再利用）。
   各辺の流量 Q_e を得る。標高は実効長 `L_eff = L·(1+α·meanE)`（`net_alpha` 既存）で高標高管を高抵抗に
   ＝流れにくく先に刈られる（ソフト忌避が流れから創発）。
3. **適応・刈り込み（Tero: dD/dt = f(|Q|) − D）**: 流量の多い辺は D 増（太る）、少ない辺は D 減。
   D < ε の辺を除去し、生き残り経路に無い孤立/行き止まりノードを prune。→「そこまでの経路を残して縮退」＝
   最外周点（と餌）を結ぶ**高コンダクタンス背骨が残り、他は退縮**。刈った分（Σ除去 D·L）は free_budget へ戻す。
4. 生き残った最外周ノードが**次の探索フェーズの新前線**になる。

**Phase 4｜前進波として移動**
後方を刈って得た体積を前線の再拡散へ回す（原形質の前送り＝translocation）。重心が外へ動き、Phase 1 へ戻る。
これを繰り返して**歩く**。

## 3. 決定性・保存則・不変条件
- **決定性**: 探索は seeded 単一 PRNG（ノード id 昇順）。Kirchhoff・刈り込みは線形代数で**乱数不要＝決定的**。
  同一 (seed, 入力列, params) → 同一 `netphys_state_hash`（ノード/辺・D・L・free を量子化して含める）。
- **保存則**: `total_mass == collected − consumed`（consumed＝標高割増等の不可逆損のみ）、非負、free_budget 超過なし。
  Phase 3 の刈り込みは管→free の保存的移動、探索は free→管。会計の漏れ（融合/刈り込み/非放射伸長）はテストで締める
  （α で非放射リークを踏んだ教訓）。
- **境界**: 全ノードが陸・範囲内（海への伸長は棄却）。**有限性**（NaN/Inf なし）。
- **ソフト標高忌避**: 探索コスト＋ `L_eff` の標高加重で、高標高帯の管は流れにくく刈られやすい（壁は作らない）。

## 4. 受け入れ条件（叩き台・headless・複数シード中央値）
既存 analysis 指標が**モデル自身の力学と評価**に昇格する点が美点。
1. **網化**: フロントが衝突する配置で、少なくとも1成分が**ループを持つ**（辺数 > ノード数−1 ＝ 冗長度>1）。
2. **餌を結ぶ**: 砂糖を2箇所以上置くと、consolidation 後に**同一連結成分**で結ばれ `flow_connected`＝true・
   有限コンダクタンス。
3. **前進波で移動**: K サイクル後にコロニー重心が初期から**有意に外へ移動**（または被覆が外周へ拡大）。
4. **効率化（Tero）**: consolidation 前後で端子間の `transport_efficiency`／総コンダクタンスが**改善**（背骨へ刈り込み）。
5. **保存則・境界・決定性・ソフト標高忌避**の不変条件。
6. **性能**: node/edge cap 内、周期 Kirchhoff の総コストが予算内。

## 5. 決める所（open knobs／実測で詰める・一部は変種）
- **周期 N**（consolidation 間隔・確定＝周期式）: 固定 N か、前線サイズに応じてスケールか（変種）。
- **融合距離 d**（anastomosis）: 塊化と過疎の間。
- **最外周の点数 k**（新成長点の数・確定＝最外周）: 重心から遠い上位 k／凸包頂点、の選定法（変種）。
- **Tero 係数**: 強化関数 `f(|Q|)`（線形／シグモイド）・減衰率・prune しきい ε。
- **保存量の担体（唯一の設計変種として要検討）**:
  - **(A) Tero 純: 質量は辺 `D·L` に載る**（最小・`flow.rs` 直再利用・決定的）＝**推奨**。
  - (B) ノード貯留＋辺コンダクタンス（過去の「体積はノード持ち」に忠実だが会計が増える）。
- **node/edge cap**（爆発防止）。
